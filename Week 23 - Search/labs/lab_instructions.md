# Instructions

## Resources

For this lab series we will use Berkeley's Pacman Projects, which are
originally described here: https://ai.berkeley.edu/search.html

However, since this version is in Python 2, we will be actually using an
equivalent implementation using Python 3, which, for your convenience, has
been included in the `pac3man` directory.

*The original source of `pac3man` is: https://github.com/jspacco/pac3man*

## Instructions

In this lab series we will focus on search problems (both uninformed and
informed ones). So, you should carefully follow all instructions found here:
https://ai.berkeley.edu/search.html

You are expected to solve all 8 tasks, focusing primarily on the first 6.

Have fun!
